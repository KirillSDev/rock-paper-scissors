### Build && Launch

<ol>

<li> Clone the repository to your local machine: </li>

```bash
    git clone
```

<li> Install Node.js version v18.12.1 using nvm: </li>

```bash
    nvm install v18.12.1
```

<li>  Install dependencies using npm: </li>

```bash
    npm install
```

<li> Start the project: </li>

```bash
   npm run start
```
